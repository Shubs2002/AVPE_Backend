### Before (Buggy)
```python

for i, character in enumerate(characters_roster):
    character['name'] = names_list[i]
    
# Problem: i goes 0, 1, 2... for successful chars only
# But names_list has ALL names including failed ones
```

### After (Fixed)
```python

    source_desc = character.get('source_description', '')
    if source_desc in names_list:
        character['name'] = source_desc
        
# Solution: Uses the original name stored in source_description
# This correctly tracks which image the character came from
```
